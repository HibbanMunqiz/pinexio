# Button Component Documentation

The **Button** component is a fully customizable button that supports multiple variants and size options. It is designed for modern, theme-aware applications that use CSS variables (defined in your global CSS) for light and dark modes.

# Props

| Prop       | Type                                             | Default   | Description                                                               |
|------------|-------------------------------------------------|-----------|---------------------------------------------------------------------------|
| `variant`  | `"primary" \| "secondary" \| "outline" \| "none"` | `"outline"` | Defines the button style variant. Uses theme-aware CSS variables.         |
| `size`     | `"xs" \| "sm" \| "md" \| "lg" \| "xl"`         | `"sm"`    | Controls the size of the button (affecting padding, font-size).           |
