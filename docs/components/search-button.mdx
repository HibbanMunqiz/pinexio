# SearchButton Component Documentations

The **SearchButton** component is a highly customizable search button that displays a command key icon (⌘K) and supports multiple size variants. It is designed for modern, theme-aware applications that use CSS variables (defined in your global CSS) for light and dark modes.

# Props

| Prop          | Type                                          | Default                      | Description                                                              |
|---------------|-----------------------------------------------|------------------------------|--------------------------------------------------------------------------|
| `placeholder` | `string`                                      | `"Search Documentation.."`   | The text displayed in the button.                                        |
| `size`        | `"xs" \| "sm" \| "md" \| "lg" \| "xl"`         | `"md"`                       | Controls the size of the button (affecting padding, font-size, icon size). |
| `...props`    | `React.ButtonHTMLAttributes<HTMLButtonElement>` | –                           | All standard button props are supported.                               |
