# Sidebar Component Documentation

The **Sidebar** component is a flexible and customizable navigation panel. It supports collapsible menus, nested links, and icons, making it ideal for documentation sites, dashboards, and applications with structured navigation.

# Props

| Prop                 | Type                                      | Default      | Description                                      |
|----------------------|-----------------------------------------|--------------|--------------------------------------------------|
| `defaultOpen`       | `boolean`                                | `false`      | Whether the sidebar is open by default.         |
| `defaultSide`       | `"left" \| "right"`                     | `"right"`    | Defines the sidebar's default position.         |
| `defaultMaxWidth`   | `number`                                 | `300`        | The default maximum width of the sidebar.       |
| `showIconsOnCollapse` | `boolean`                              | `true`       | Whether icons should be visible when collapsed. |
| `...props`          | `React.HTMLAttributes<HTMLDivElement>`   | â€“            | Supports all standard `div` attributes.         |
